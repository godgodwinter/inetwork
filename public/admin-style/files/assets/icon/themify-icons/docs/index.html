Themify Icons Plugin

Themify Icons Plugin is a plugin based on the Themify Icons font that allows you to insert buttons and icons in the post content, menus, and widget titles.

How To Use It

The plugin is very simple to use. Once you activate the plugin, you can easily use it to do the following things:

Inserting icon buttons in the content:

  1. Create new post/page from post/page > Add New
  2. In post editor, click on the Themify smiley icon and you will see a pop up with options to customize the button styling, text and link

Inserting Themify Icons in WordPress Menus:

  1. Go to Appearance > Menus
  2. Select the Menu you want to edit
  3. Then expand the menu item and click on the "insert icon" button

Inserting Themify Icons in widget titles:

  1. Go to Appearance > Widgets
  2. Expand the widget options and click on the "insert icon" button
